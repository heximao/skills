# 基本面分析框架

## 估值指标

### PE（市盈率）
**公式**：股价 / 每股收益（EPS）

| 类型 | 偏低 | 合理 | 偏高 |
|------|------|------|------|
| 成长股 | < 15倍 | 15-35倍 | > 50倍 |
| 价值股 | < 8倍 | 8-15倍 | > 20倍 |

**注意**：
- PE需与行业均值对比，不同行业差异极大
- 周期股在业绩高点时PE反而低，需警惕
- 负PE或极高PE可能是成长股或亏损股

### PB（市净率）
**公式**：股价 / 每股净资产

| PB值 | 含义 |
|------|------|
| < 1倍 | 股价低于净资产，可能被低估或资产质量差 |
| 1-3倍 | 合理区间 |
| > 5倍 | 偏高，需关注资产质量和成长性 |

**适用**：
- 重资产行业（银行、地产、制造业）
- 不适用：轻资产、高成长公司

### ROE（净资产收益率）
**公式**：净利润 / 净资产

| ROE值 | 含义 |
|-------|------|
| < 8% | 盈利能力较弱 |
| 8-15% | 正常水平 |
| > 20% | 优秀水平，持续高ROE是优质公司特征 |

**杜邦分析**：
ROE = 净利润率 × 资产周转率 × 权益乘数
- 高净利润率：产品定价能力强
- 高资产周转率：运营效率高
- 高权益乘数：财务杠杆大（注意风险）

## 财务健康度检查

### 营收增速
| 指标 | 标准 |
|------|------|
| 同比增长 | 持续正增长为佳 |
| 季度环比 | 观察趋势是否稳定 |
| 3年复合增长率 | > 10%为良好 |

### 净利润率
| 指标 | 标准 |
|------|------|
| 净利润率 | 行业领先为佳 |
| 趋势 | 维持稳定或提升 |
| 与同行对比 | 高于行业平均 |

### 负债率
| 指标 | 标准 |
|------|------|
| 资产负债率（非金融） | 一般 < 60%为安全 |
| 有息负债率 | 越低越好 |
| 流动比率 | > 1为安全 |
| 速动比率 | > 0.8为安全 |

**注意**：金融企业（银行、保险）负债率天生较高，需用不同标准

### 现金流
| 指标 | 标准 |
|------|------|
| 经营性现金流 | 持续为正为佳 |
| 经营现金流/净利润 | > 1为佳（利润含金量高） |
| 自由现金流 | 持续为正为佳 |

### 毛利率趋势
| 趋势 | 含义 |
|------|------|
| 上升 | 产品竞争力强或成本控制好 |
| 稳定 | 业务模式成熟 |
| 下降 | 竞争加剧或成本上升，需警惕 |

## 行业对比

### 高PE行业（通常）
- 科技、医药、消费
- 成长性高，市场给予估值溢价

### 低PE行业（通常）
- 银行、地产、公用事业
- 成长性低，估值回归

### 周期性行业
- 钢铁、煤炭、有色、化工
- PE低时可能是业绩高点（风险）
- PE高时可能是业绩低点（机会）

## 成长性评估

### PEG指标
**公式**：PE / 净利润增长率

| PEG值 | 含义 |
|-------|------|
| < 1 | 可能被低估 |
| = 1 | 估值合理 |
| > 1 | 可能被高估 |

### 成长股特征
- 营收增速 > 20%
- 净利润增速 > 20%
- 行业处于成长期
- 市场份额提升

## 价值股特征

### 安全边际
- PE/PB处于历史低位
- 股息率 > 3%
- 现金流稳定
- 资产质量好

### 风险控制
- 负债率合理
- 业务稳定
- 管理层持股
- 分红稳定

## 财务造警报红信号

1. **净利润持续增长但经营现金流为负**
2. **应收账款增速远超营收增速**
3. **存货周转率异常下降**
4. **毛利率远超行业平均水平**
5. **频繁并购且商誉占比高**
6. **关联交易频繁且金额大**
