# 技术指标解读规则

## MACD 信号

### 基本概念
- **DIFF线**：快线（DIF），一般为12日EMA减26日EMA
- **DEA线**：慢线（MACD线），DIFF的9日EMA
- **MACD柱**：2×(DIFF - DEA)，红柱为正，绿柱为负

### 交易信号
| 信号 | 含义 | 操作建议 |
|------|------|----------|
| 金叉（DIFF上穿DEA） | 短期上涨信号 | 可考虑买入 |
| 死叉（DIFF下穿DEA） | 短期下跌信号 | 注意风险 |
| MACD柱由负转正 | 动能由跌转涨 | 关注买入机会 |
| MACD柱由正转负 | 动能由涨转跌 | 注意卖出信号 |
| 顶背离 | 价格创新高但MACD未创新高 | 看空，注意回调 |
| 底背离 | 价格创新低但MACD未创新低 | 看多，关注反弹 |

### 判断强度
- 金叉在0轴下方：反弹信号，强度一般
- 金叉在0轴上方：强势信号，强度较强
- 背离信号：比交叉信号更可靠

## RSI 信号

### 超买超卖区间
| RSI值 | 状态 | 操作建议 |
|-------|------|----------|
| > 80 | 超买区 | 注意回调风险，谨慎追高 |
| 70-80 | 偏强 | 谨慎追高，设好止损 |
| 30-70 | 正常波动 | 观察趋势，正常交易 |
| 20-30 | 偏弱 | 可能接近底部，关注反弹 |
| < 20 | 超卖区 | 关注反弹机会，分批建仓 |

### RSI背离
- **顶背离**：股价创新高但RSI未创新高 → 卖出信号
- **底背离**：股价创新低但RSI未创新低 → 买入信号

## KDJ 信号

### 基本概念
- **K值**：快线，反映当前价格相对位置
- **D值**：慢线，K值的平滑
- **J值**：3K - 2D，反应更敏感

### 交易信号
| 信号 | 条件 | 操作建议 |
|------|------|----------|
| 金叉 | K上穿D，尤其在20以下 | 买入信号，强势买入 |
| 死叉 | K下穿D，尤其在80以上 | 卖出信号，强势卖出 |
| 超买 | J > 100 | 短期回调概率大 |
| 超卖 | J < 0 | 可能出现反弹 |

### KDJ与RSI配合
- KDJ金叉 + RSI < 30：买入信号更强
- KDJ死叉 + RSI > 70：卖出信号更强

## 布林带信号

### 基本概念
- **中轨**：20日移动平均线
- **上轨**：中轨 + 2倍标准差
- **下轨**：中轨 - 2倍标准差

### 交易信号
| 信号 | 含义 | 操作建议 |
|------|------|----------|
| 价格突破上轨 | 强势突破或超买 | 观察成交量，判断真突破 |
| 价格触碰下轨 | 支撑位或超卖 | 关注反弹机会 |
| 带宽收窄 | 波动变小 | 即将出现大波动，准备方向选择 |
| 带宽扩张 | 波动变大 | 趋势确立，跟随趋势 |

### 布林带配合
- 价格在上轨附近 + 放量突破：看多
- 价格在下轨附近 + 缩量：看反弹

## 均线系统

### 常用均线
| 均线 | 周期 | 含义 |
|------|------|------|
| MA5 | 5日 | 短期趋势，周内波动 |
| MA10 | 10日 | 短期趋势，两周内波动 |
| MA20 | 20日 | 中短期趋势，月内波动 |
| MA60 | 60日 | 中期趋势，季内波动 |
| MA120 | 120日 | 长期趋势，半年内波动 |
| MA250 | 250日 | 年线，重要长期支撑/压力位 |

### 均线形态
| 形态 | 条件 | 含义 |
|------|------|------|
| 多头排列 | 价格 > MA5 > MA10 > MA20 > MA60 | 趋势强势，持有 |
| 空头排列 | 价格 < MA5 < MA10 < MA20 < MA60 | 趋势弱势，谨慎 |
| 黄金交叉 | MA5上穿MA20 | 中期买入信号 |
| 死亡交叉 | MA5下穿MA20 | 中期卖出信号 |
| 年线支撑 | 价格回调至MA250附近企稳 | 重要支撑位 |
| 年线压力 | 价格反弹至MA250附近受阻 | 重要压力位 |

### 均线配合
- 短期均线（MA5/MA10）：适合短线交易
- 中期均线（MA20/MA60）：适合波段操作
- 长期均线（MA120/MA250）：适合长期投资判断

## 综合技术分析

### 多指标共振
当多个技术指标同时发出相同信号时，信号更可靠：
- MACD金叉 + KDJ金叉 + RSI从超卖区回升：强势买入信号
- MACD死叉 + KDJ死叉 + RSI从超买区回落：强势卖出信号

### 指标矛盾处理
当指标信号矛盾时：
1. 以趋势类指标（均线系统、MACD）为主
2. 超买超卖类指标（RSI、KDJ）为辅
3. 结合成交量和价格形态判断
